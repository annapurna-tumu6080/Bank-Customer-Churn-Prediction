# Bank-Customer-Churn-Prediction
Developed a Bank Customer Churn Prediction model using Machine Learning to identify customers likely to leave the bank. Performed data preprocessing, EDA, feature encoding, and trained Logistic Regression and Random Forest models. Evaluated performance using Accuracy, Confusion Matrix, Precision, Recall, and F1-Score.
## Project Overview
This project predicts whether a bank customer is likely to leave the bank (churn) based on customer demographics, account information, and banking behavior. The objective is to help banks identify at-risk customers and improve customer retention strategies.

## Dataset Features
- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Membership Status
- Estimated Salary

### Target Variable
- Exited
  - 0 = Customer Stays
  - 1 = Customer Leaves

Random Forest Classifier outperformed Logistic Regression and achieved an accuracy of 86.65%, making it the best-performing model for customer churn prediction.
